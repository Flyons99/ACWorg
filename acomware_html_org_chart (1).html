Konfigurace GitHub

Nastavení GitHub repository

Jak získat GitHub Personal Access Token:

  1. Jděte na GitHub.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. Klikněte "Generate new token (classic)"
  3. Nastavte název (např. "Acomware Org Chart")
  4. Vyberte oprávnění: repo (Full control of private repositories)
  5. Klikněte "Generate token" a zkopírujte token
  6. Vytvořte repository pro ukládání dat (např. "acomware/org-chart")
